# Security Policy

This is a static personal site (Jekyll on GitHub Pages) — no backend, no user data, no auth.

## Reporting a vulnerability

If you find something that looks like a security issue (XSS in a post, leaked secret in the repo, broken redirect, etc.), please report it privately through one of:

- GitHub private security advisory — <https://github.com/melipass/melix/security/advisories/new>
- LinkedIn DM — <https://www.linkedin.com/in/melissa-silva-s/>

Please don't open a public issue for security reports.

## Scope

In scope: this repository and the deployed site at <https://melix.cl>.

Out of scope: third-party services I link to from posts, and the upstream Chirpy theme (report those at <https://github.com/cotes2020/jekyll-theme-chirpy/security>).
